# Санжар & Айзат — тойға шақыру сайты

Одностраничный свадебный сайт-приглашение, готовый для публикации через GitHub + Vercel.

## Как посмотреть локально

Откройте файл `index.html` в браузере.

## Что уже готово

- первый экран с именами молодожёнов;
- дата и время: 22 августа 2026, 19:00;
- текст приглашения на казахском языке;
- живой таймер до свадьбы;
- блок адреса ресторана DoubleBB и кнопка 2GIS;
- блок dress code;
- кнопка подтверждения участия;
- финальный экран;
- кнопка фоновой музыки;
- адаптация под телефон.

## Что нужно добавить перед публикацией

### 1. Музыка

Положите ваш законно полученный MP3-файл в папку:

`assets/music/aygyt-aq-koilek.mp3`

После этого кнопка музыки на сайте заработает.

### 2. Подтверждение участия

Создайте Google Form либо предоставьте WhatsApp-номер для RSVP.
Затем в файле `script.js` замените:

```js
const RSVP_URL = "";
```

на ссылку, например:

```js
const RSVP_URL = "https://forms.gle/...";
```

### 3. Подтвердить дресс-код

Сейчас выбрана нейтральная палитра: молочный, бежевый, коричневый и чёрный.
Её можно заменить после согласования с молодожёнами.

## Размещение

На следующем шаге проект можно загрузить в GitHub и задеплоить бесплатно через Vercel.


## Обновление: WhatsApp и галерея

Кнопка подтверждения сейчас ведёт в WhatsApp на номер `+7 777 801 8143` с заранее подготовленным сообщением гостя.

Чтобы вместо WhatsApp использовать Google Form:
1. Создайте форму по шаблону из файла `google-form-questions.txt`.
2. Скопируйте ссылку Google Form.
3. В файле `script.js` замените значение строки `const RSVP_URL = "...";` на ссылку формы.
